Build Installer
Package Meeting Recorder as a native desktop app for macOS or Windows.
🍎 macOS
🪟 Windows 11
How it works

electron-builder produces a .dmg drag-and-drop installer and a .pkg for Apple Silicon (arm64).

Python and Whisper are not bundled — users need python3 and openai-whisper installed. Run ./setup.sh first.

Step-by-step
1
Run setup script (first time only)
chmod +x setup.sh && ./setup.sh
2
Install electron-builder
npm install --save-dev electron-builder
3
Add app icon (optional)
Place a 1024×1024 PNG as assets/icon.png, then convert it:
mkdir assets/icon.iconset sips -z 16 16 assets/icon.png --out assets/icon.iconset/icon_16x16.png sips -z 512 512 assets/icon.png --out assets/icon.iconset/icon_512x512.png cp assets/icon.png assets/icon.iconset/icon_512x512@2x.png iconutil -c icns assets/icon.iconset -o assets/icon.icns
4
Build DMG + PKG installer
npm run build
5
First launch on another Mac (unsigned app)
Right-click → Open → Open, or remove the quarantine flag:
xattr -cr /Applications/Meeting\ Recorder.app
Output files

dist/Meeting Recorder-1.0.0-arm64.dmg
dist/Meeting Recorder-1.0.0-arm64.pkg
dist/mac-arm64/Meeting Recorder.app

Overview

electron-builder produces an NSIS installer (.exe) and a portable .exe for Windows x64. Cross-compilation from macOS is possible but requires wine; building on a Windows machine is simpler.

Python, Whisper, and ffmpeg are not bundled. Run setup-windows.ps1 first to install all prerequisites.

Prerequisites (PowerShell — run as Administrator)
1
Allow script execution (once per machine)
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
2
Run the setup script
Installs Chocolatey, Node.js, Python 3, ffmpeg, VB-Audio Virtual Cable, openai-whisper, and npm dependencies.
.\setup-windows.ps1
3
Configure audio loopback
After the VB-Audio driver installs, set your meeting app (Teams/Zoom) to output to CABLE Input. Meeting Recorder will capture from CABLE Output.
Building the installer
4
Install electron-builder (if not done by setup script)
npm install --save-dev electron-builder
5
Add a Windows icon (optional)
Place a 256×256 .ico file at assets/icon.ico. Online converters can create .ico from a PNG.
6
Build NSIS installer + portable
npm run build:win
7
Build portable .exe only
npm run build:win:portable
Output files

dist/Meeting Recorder Setup 1.0.0.exe — NSIS installer
dist/Meeting Recorder 1.0.0.exe — portable (no install)

SmartScreen may warn on first launch of an unsigned binary. Click "More info → Run anyway", or sign the binary with a code-signing certificate.
